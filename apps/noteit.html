核心功能

为日记爱好者精心设计的每一个细节

  • 🤖

    AI 智能摘要

    一键生成今日回顾，AI 帮你提炼情绪与关键事件，让回忆更清晰。

  • 🔒

    本地隐私存储

    所有日记存储在本地，不上传服务器，Face ID / Touch ID 保护，你的故事只属于你。

  • 📅

    时间轴回顾

    按日、月、年浏览你的记录，「一年前的今天」功能让你随时重温过去。

  • 🌙

    深色模式

    完美适配系统深色模式，夜间书写更护眼，专注写作不受干扰。

  • 🔍

    全文搜索

    快速定位任意一条记录，支持关键词搜索，找到你想要的那一刻。

  • ☁️

    iCloud 同步

    可选 iCloud 在 iPhone、iPad、Mac 间无缝同步，多端无缝续写。

  • 🏷️

    标签与分类

    为日记打标签，轻松归类工作、生活、旅行等不同场景。

  • 📊

    写作统计

    追踪连续打卡天数、总字数、情绪分布，见证自己的成长轨迹。

常见问题

核心功能完全免费，包括无限日记、本地存储和全文搜索。高级 AI 功能和主题皮肤可选购。

所有数据存储在本地设备，不上传到任何服务器。支持 Face ID / Touch ID 锁定。iCloud 同步为可选功能，数据始终由你掌控。

Noteit 支持 iOS 16+、iPadOS 16+ 和 macOS 13+，一次购买全平台畅用。

部分 AI 功能需要网络连接，但所有基础写作和存储功能在离线状态下均可正常使用。

支持导出为 PDF 或纯文本格式，方便备份或与他人分享。
